# Fitness Tracker API

# Project Description
The Fitness Tracker API is a backend application built with Django and Django REST Framework.  
It allows users to track their workouts, exercises, and fitness progress through a RESTful API.

This project is part of my backend learning journey and focuses on API design, database modeling, and CRUD operations.

#Features
- User registration and authentication
- Create, read, update, and delete workouts
- Track exercises performed during workouts
- Store workout duration and dates
- RESTful API endpoints
- JSON-based responses
